# 🎯 Talento Top Empleo

Una plataforma moderna de búsqueda de empleo enfocada en Colombia y Latinoamérica.

## 🚀 Características

- ✅ Búsqueda de empleos en tiempo real
- ✅ Interfaz moderna y responsive
- ✅ Múltiples ofertas de empleo
- ✅ Formulario de contacto funcional
- ✅ Optimizado para SEO
- ✅ Rápido y eficiente con Next.js

## 📋 Requisitos Previos

- Node.js 16.x o superior
- npm o yarn

## 🔧 Instalación Local

1. **Clonar el repositorio:**
```bash
git clone https://github.com/valdesjulian975-beep/talento-top-empleo.git
cd talento-top-empleo
```

2. **Instalar dependencias:**
```bash
npm install
```

3. **Ejecutar el servidor de desarrollo:**
```bash
npm run dev
```

4. **Abrir en el navegador:**
```
http://localhost:3000
```

## 📦 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run start` - Inicia el servidor de producción
- `npm run lint` - Ejecuta el linter

## 🌐 Desplegar en Vercel

### Opción 1: Desde la CLI

```bash
npm install -g vercel
vercel
```

### Opción 2: Desde GitHub

1. Ve a [Vercel Dashboard](https://vercel.com/dashboard)
2. Haz clic en "New Project"
3. Conecta tu repositorio de GitHub
4. Vercel detectará automáticamente que es un proyecto Next.js
5. Haz clic en "Deploy"

## 🛠️ Stack Tecnológico

- **Next.js 14** - Framework React
- **React 18** - Librería UI
- **Tailwind CSS** - Estilos
- **JavaScript** - Lenguaje principal
- **PostCSS** - Procesador CSS
